Context

BI development is a key process in analysts teams to create data-informed solutions. Its cross-functional nature makes it a central domain in the

Problem

While most companies rely on tools from vendors such as Microsoft or Salesforce for their BI solutions, solo entrepreneurs and small businesses might prefer less expensive solutions that are not tied to a vendor ecosystem.

Solution

Several frameworks in the python ecosystem are targeted towards data app development such as Streamlit and Reflex. These frameworks enable developers to create dashboards by relying only on the python language. Under the hood, these frameworks usually translate python code to HTML/React components.

Process

The first step was defining the metrics to display and the different types of charts to use. Once the metrics were defined, data from the source file was processed for each metric.
